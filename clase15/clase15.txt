¿Por qué un lenguaje de programación sólo puede utilizarse en algunos sistemas operativos y en otros no?. 

Por que cada sistema esta hecho de maneras diferentes a otros, sus estructuras u arquitectura es distinta. Entonces ciertos lenguajes estan preparados para correr y ser ejecutados para ciertos sistemas porque comparten la misma arquitectura y forma de ser. 
Del lado del back end esta el IDE tiene la capacidad de compilar o crear un archivo ejecutable para un tipo puntual de procesador. Pasar de codigo a un archivo creado ejecutable, si la sintaxis esta bien el archivo se ejecuta.
Del lado del front end esta el 	VSC que es un editor de texto avanzado, que por si mismo no puede compilar por lo cual le hace falta un interprete, ese programa lo interpreta lo que quiere el usuario y crea un archivo ejecutable que lo hace funcionar en cualquier lado, apoyado por el motor v8.
En el IDE  importa el hardware, y el VSC no importa pero si necesita un interprete.


¿Qué tipo de máquina virtual soporta virtualBox?. 

Puede ejecutar tres o cuatro máquinas virtuales básicas en un host que tenga 4 GB de RAM, aunque se necesitarán más recursos si se desea ejecutar más.
Algunas de la mas instaladas son GNU/Linux, Mac OS X, OS/2 Warp, Microsoft Windows, y Solaris/OpenSolaris. 


¿Qué función cumple el hypervisor en la virtualización?

Hypervisor es el encargado de repartir todo el hardawre que le da el kernel para repartir en todas las maquinas virtuales, entonces entre mas maquinas virtuales tenga mas servicio se va consumir pero mas tareas se podran hacer en especifico. Y todo lo q susece en la maquina virtual se queda ahí y no afecta a la pc real. Es un sistema de pruebas o matrix.

La virtualizacion: es agarrar las cosas de hardware y con el software llevarlo a otro nivel.
Con la virtualizacion podemos dividir un disco en dos. Osea un fisico se divide en dos logicos y funciona mas rapido para sacarle mas provecho al hardware medinate el software.
