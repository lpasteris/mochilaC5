Git es un proyecto de código abierto maduro desarrollado originalmente por Linus Torvalds, creador del kernel del sistema operativo Linux, en 2005.

GitHub es una plataforma de desarrollo colaborativa de software, un sistema de control de versiones creado originalmente por Linus Torvalds para controlar el desarrollo del kernel de Linux
