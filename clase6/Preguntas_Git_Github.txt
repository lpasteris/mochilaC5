¿Quién inventó el sistema de control de versión de Git y por qué?
respuesta: Git es un proyecto de código habierto desarrollado originalmente por Linus Torvlads,
	   el creador del kernel de Linux. Git nació cuando el gestor de paquetes de Linux pasó 
	   a ser de pago.

	  con el enfoque al desarrollo, haciendo cada vez más hicapie en que el núcleo de la empresa
	  lo forman los desarrolladores.
